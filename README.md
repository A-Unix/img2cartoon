# img2cartoon
A basic tool to convert any image into cartoon with embedding the song in video.

# Installation Instructions

## Clone the repo
```bash
git clone https://github.com/Atuls-git/img2cartoon.git
```
## Move to cloned folder
```bash
cd img2cartoon
```
## Grant permissions
```bash
chmod +x *
```
## Install requirements
```bash
./setup.sh
```
## Run main file
```bash
python3 main.py
```

Enjoy :)
